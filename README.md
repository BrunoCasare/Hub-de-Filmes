# Catálogo de filmes

## Projeto testando MySQL e PHP

https://casaflix.site.je

Site feito como trabalho de faculdade na matéria de Programaçâo Web. Hospedado na infinityfree.  

Tecnologias usadas:  
-PHP  
-MySQL  

CRUD básico.  
Para cadastrar um filme, basta clicar no botão "Cadastrar" no canto superior direito, após isso preencha o formulário com nome, nota, gênero e capa do filme. Ao confirmar, o site voltará para a página principal e o filme estará a mostra juntamente com um código de identificação.  
  
Para editar um filme já cadastrado, basta clicar no botão "Editar" no canto superior direito, após isso preencha o formulário com o código de identificação do filme e preencher com as novas informações e confirmar.
  
Para excluir um filme, basta clicar no botão "Excluir" no canto superior direito, após isso preencha o formulário com o código de identificação do filme e confirmar.
